## Project setup

How to open a project

### Clone project

```
git clone git@github.com:yukos1221/ecwid-pls-hire-me.git
```

### Got to folder

```
cd ecwid-pls-hire-me 
```

### Install dependencies

```
npm ci
```

### Run development server

```
npm run dev
```

The project will be launched on URL 

```
http://localhost:5173
```
